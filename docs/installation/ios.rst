napalm-ios dependencies
=======================


Ubuntu and Debian
-----------------

.. code-block:: bash
  
  sudo apt-get install -y --force-yes libssl-dev libffi-dev python-dev python-cffi

RedHat and CentOS
-----------------

.. code-block:: bash

    sudo yum install -y python-pip gcc openssl openssl-devel libffi-devel python-devel
